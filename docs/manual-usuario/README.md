# Manual de usuario de esdata

## Objetivo

Este directorio contiene el manual vivo de `esdata`.

No es la fuente de estado del proyecto. Para fase activa, estado ejecutivo y siguiente paso exacto, la fuente activa sigue siendo `../master-execution-roadmap.md`.

Este manual explica:

- que hace `esdata`
- que superficies de uso existen
- como usar la herramienta
- que incluye hoy
- que limites y exclusiones tiene
- que aspectos tecnicos necesita conocer un operador o integrador

## Regla de mantenimiento

- El manual se mantiene por capitulos pequenos para reducir colisiones entre agentes.
- Cada agente debe actualizar solo el capitulo afectado por su cambio.
- Si hace falta un capitulo nuevo, se crea aqui y se enlaza desde este indice.
- Los cambios de codigo o comportamiento visible no deben cerrarse sin revisar el manual correspondiente.
