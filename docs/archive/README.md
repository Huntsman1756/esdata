# Archive

Carpeta de documentacion historica del repo.

## Regla

- No leer esta carpeta por defecto.
- La fuente activa unica de estado y ejecucion es `../master-execution-roadmap.md`.
- Los documentos archivados sirven para auditoria, contexto y trazabilidad, no para decidir trabajo actual.

## Subcarpetas

- `handoffs/` — handoffs y snapshots de sesion
- `status/` — estados puntuales de produccion o sesion ya superados
- `plans/` — roadmaps y planes cerrados o sustituidos
- `infra/` — handoffs o notas historicas de infraestructura
- `postmortems/` — postmortems cerrados
